hello guys. this is a description on how to run this django project.

- first, ensure that your have python installed on your computer
- create a virtual environment by running this command 'python -m venv environmentname' or rather 'python3 -m venv environmentname' replacing 3 with your python version.
- activate the virtual environment after creating by running 'source environmentname/bin/activate' for UNIX(linux, mac os) users or 'cd environmentname/Script/activate.bat' in windows.
- after activation of the virtual environment, run go to the directory of your project using the terminal.
- once inside the folder of your project, run 'python manage.py runserver' to get the project up and running.
- to break the server, press ctrl + c.


- that's it. enjoy.
